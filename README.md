# Arabic Lexicon Atelier

Веб-приложение для изучения классического арабского языка с помощью карточек, категорий и интерактивных тестов.

## Возможности

- главная страница с описанием проекта;
- каталог карточек с поиском и фильтрацией по категориям;
- просмотр подробной информации о карточке;
- создание, редактирование и удаление карточек;
- карточки с визуальным переворотом;
- тест с вариантами ответов и подсчётом результата;
- страница результатов теста;
- административная панель Django;
- демонстрационное заполнение базы данных.

## Технологический стек

- Python 3.11+
- Django 5.2
- SQLite
- Bootstrap 5
- Django Templates
- Django Forms

## Установка и запуск

1. Создайте виртуальное окружение:

```bash
python -m venv .venv
```

2. Активируйте окружение:

Linux/macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Примените миграции:

```bash
python manage.py migrate
```

5. Загрузите демонстрационные данные:

```bash
python manage.py seed_data
```

6. При необходимости создайте администратора:

```bash
python manage.py createsuperuser
```

7. Запустите сервер:

```bash
python manage.py runserver
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:8000/`.

## Демонстрационный сценарий

1. Откройте главную страницу и ознакомьтесь с описанием проекта.
2. Перейдите в раздел карточек и воспользуйтесь поиском или фильтром по категории.
3. Откройте карточку, изучите пример и проверьте визуальный переворот.
4. Добавьте новую карточку через форму.
5. Запустите тест и посмотрите результаты.
